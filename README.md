# Project 5 - Loan-Management-System
Developed a Loan Management System using ServiceNow to automate loan requests, eligibility validation, approvals, and notifications through Flow Designer.

Project Overview:

This system streamlines the loan request process by validating eligibility, managing approval workflows, and sending automated notifications using Flow Designer.

Key Features:
- Service Catalog-based loan request form
- Automated eligibility validation (salary vs loan amount)
- Manager approval workflow using Flow Designer
- Auto rejection for high-risk requests
- Real-time status updates
- Email notifications to users

Workflow:
- User submits loan request via Service Portal
- System checks eligibility criteria
- Eligible requests go for manager approval
- Request is approved/rejected
- Final status is updated and notified

Tools Used: 
- ServiceNow
- Flow Designer
- Service Catalog
- Notifications
